# BookNest 📚

A beautifully designed mobile app for discovering books, organizing reading lists, and tracking reading progress — powered by Open Library.

## 🚀 Features

- 🔍 Search books by title or author
- 📖 View detailed book metadata
- 🗂 Organize books into shelves (Want to Read, Reading, Finished)
- 📊 Track reading progress and notes
- ⚡ Offline-first experience with local persistence
- 🧠 Smart “Continue Reading” and “Recently Viewed” sections

## 🧠 What this project demonstrates

- API integration (Open Library)
- State management with Zustand
- Avoiding infinite render loops (optimized selectors)
- Offline persistence using AsyncStorage
- Clean architecture with feature-based structure
- Performance optimization with memoization
